Coffee-2-Go

This was my final independent project for the General Assembly course. The idea behind Coffee-2-Go is to avoid the dreaded rush-hour line at the coffee shop. You can search for coffee shops using google maps, pick one and order your coffee online, and go pick it up.

The Back End is Ruby on Rails with PostgreSQL for the database which has a complex structure using 6 different models. The Front End is Angular.js incorporating the Google Places API along with Flexbox for CSS styling. User authentication is done with JWT tokens.

NitPic

NITPIC is a social media / photo sharing app. Create photo albums, upload and leave comments on photos.

This was a collaborative project for the General Assembly course. My role was to build out the Angular.js Front End and the Back End Rails API. My team mates were responsible for styling the app and incorporating Paperclip for photo upload.

The Store Front

The Store Front is exactly that - a store front app. Browse products by category or use the search feature for something specific. Add products to the cart and checkout when you're ready.

This was a collaborative project for the General Assembly course. My role was to build the Angular.js Front End and the Back End using Node, Express, and MongoDB with Mongoose. My team mates were responsible for styling the app and database support.

DogVentures

A social media app for dog-lovers to share information about dog-friendly travel locations. Users can write reviews for specific locations. Reviews include a comment section for anyone to join in the conversation.

This was an independent project for the General Assembly course created with Node.js, Express.js, Handlebars for the view engine and MongoDB for the database. I used Bootstrap 3 for CSS styling and Passport.js for User Authentication.
